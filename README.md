# AssistenteFinanceiroFJ

Trabalho desenvolvido pelos alunos João Vitor Barbosa e Fillip Pereira de Sistemas de Informação na disciplina de Interação Humano Computador da PUC-MG Barreiro.
Desenvolvido em C# o projeto não depende de nenhuma dependência específica, somente é necessário não apagar o arquivo de texto que contém o extrato dos lançamentos feitos.
O trabalho foi desenvolvido primeiramente para atender os seguintes requisitos:
- O usuário poderá inserir uma renda. 
- O usuário poderá inserir um gasto. 
- O usuário poderá visualizar o extrato e o saldo.

Após realizar uma coleta de dados, foi proposta uma intervenção para inclusão de uma nova funcionalidade para simulação de investimentos.
